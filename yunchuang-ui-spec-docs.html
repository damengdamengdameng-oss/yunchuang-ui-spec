弹框

弹框标题推荐 14px，正文 14px，底部按钮遵循本规范中的按钮尺寸。弹出框圆角统一为 6px；在无蒙版场景下，阴影统一使用 0px 8px 32px 0px rgba(0, 0, 0, 0.16)。

新建教材
弹框标题使用 14px，正文使用 14px。底部主要按钮建议使用渐变 M，取消按钮使用白色 M。
